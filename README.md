# Pokémon Battle Calculator MVP (KR) — Static Web App
MIT License

## 구성
- `index.html` — 단일 페이지 앱 (탭: 대미지 계산기 / 쇼다운 로그 번역기 / 수동 배틀 러너)
- `style.css` — 미니멀 UI
- `app.js` — 핵심 로직(대미지 공식, 타입 상성, 로그 번역, 수동 러너)
- `assets/sample.json` — 샘플 입력

## 사용 방법
1) 폴더를 GitHub에 올리거나, `index.html`을 더블클릭하여 로컬에서 바로 실행해도 작동합니다(브라우저만 있으면 됨).
2) **대미지 계산기** 탭에서 공격측/수비측 정보를 입력하고 기술을 선택하면 좌측 하단에 '최소~최대' 대미지, OHKO/2HKO 확률, 한국어 로그가 출력됩니다.
3) **쇼다운 로그 번역기** 탭에 원문 로그를 붙여넣고 [번역] 버튼을 누르면 한국어 로그로 변환됩니다.
4) **수동 배틀 러너** 탭에서 간단한 명령(예: `ATTACK 이름 기술명`)을 입력하면 실제 게임풍 로그가 생성됩니다.

## 대미지 공식(Gen 6+ 기반)
```
Damage = floor(floor(floor(((2*L/5 + 2) * P * A / D) / 50) + 2) * Mod) * r
- L: 레벨, P: 위력, A/D: 실능(긍/부성격, EV/IV, 도구/특성 반영), r: 0.85~1.00(16스텝)
- Mod: STAB, 타입상성, 급소, 화면(리플렉터/빛의장막), 날씨, 화상(물리 0.5, 근성 예외), 기타 배수
```
실전 적용이 필요한 요소는 `추가 배수`(공격측·수비측·기술별) 필드에 배수로 적어 임시로 대응할 수 있습니다.
